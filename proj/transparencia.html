Transparência Bot

BOT DE AUTOMAÇÃO WEB · PORTAL DA TRANSPARÊNCIA GOV.BR
// sobre o projeto

Bot de automação web (RPA) para consulta de pessoas físicas no Portal da Transparência do Governo Federal. Desenvolvido como desafio técnico para vaga de Full Stack Developer Python. Realiza consultas automatizadas, extrai dados de benefícios e retorna evidência em Base64.

// funcionalidades
  • Bypass de WAF AWS CloudFront com Chromium headless + anti-detecção via add_init_script
  • Navegação por DOM assíncrono usando aria-controls para localizar accordion de benefícios
  • Captura de evidência em Base64 com accordion expandido e banner LGPD removido
  • API REST com FastAPI e documentação Swagger automática
  • Suporte a busca por nome, CPF e NIS com detecção automática do tipo de identificador
  • Orquestração completa via n8n: Webhook → API → Google Drive → Google Sheets
  • Containerização com Docker Compose — bot API (8001) + n8n (5678)
  • Tratamento de erros específicos por tipo de identificador e timeout do portal
// stack técnico
Python 3.12 FastAPI Playwright Docker n8n Google Drive API Google Sheets API
